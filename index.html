Claude

Anthropic製。長文の理解・要約・執筆が得意。細かいニュアンスや倫理的な判断に優れており、論文や報告書レベルの文章を生成できる。

// 代表ツール
Claude.ai Claude API
// こんな時に使う
  • 長い文書の要約・翻訳
  • メールや報告書の下書き
  • アイデアのブレスト相手
  • 複雑な質問への丁寧な回答

ChatGPT

OpenAI製。最も有名な会話AIで、汎用性が高く何でもこなす万能選手。プラグインや画像生成(DALL·E)も統合されていて、初心者にも使いやすい。

// 代表ツール
ChatGPT GPT-4o OpenAI API
// こんな時に使う
  • 日常的な質問・調べもの
  • 文章の校正・リライト
  • 料理レシピや旅行計画
  • 画像の説明・分析

Gemini

// 代表ツール
Gemini NotebookLM
// こんな時に使う
  • 最新ニュースの要約
  • Gmailの返信下書き
  • Googleスプレッドシートの自動化
  • YouTube動画の要約

画像生成

Midjourney

クオリティと芸術性で世界トップクラス。テキストから驚くほどリアルで美しいアート作品を生成。プロのデザイナーやクリエイターも愛用している。

// 代表ツール
Midjourney v7 Discord Bot
// こんな時に使う
  • アート作品・イラストの制作
  • 広告バナーやビジュアル素材
  • キャラクターデザインのアイデア出し
  • インテリアや建築のビジュアライズ

画像生成

Adobe Firefly

Adobe製で商用利用に強い。Photoshopなどのツールにも統合されており、既存の画像を拡張する「ジェネレーティブ塗りつぶし」が特に便利。

// 代表ツール
Adobe Firefly Photoshop AI DALL·E 3
// こんな時に使う
  • 写真の背景を差し替える
  • 商用OKな素材を素早く作る
  • 既存画像を自然に拡張する
  • テキストから写真風の画像生成
難易度
無料あり

動画生成

Runway

テキストや画像から動画を生成するAI。映画的な映像表現が可能で、ハリウッドの映像制作現場でも使われ始めている。動画編集機能も豊富。

// 代表ツール
Runway Gen-3 Sora Kling
// こんな時に使う
  • SNS用のショート動画制作
  • 商品紹介動画のプロトタイプ
  • 映像コンテンツのアイデア出し
  • 静止画をアニメーション化する
難易度
無料あり
🎭
動画生成

HeyGen

AIアバターが自然に話す動画を自動生成。顔や声をクローンする機能もあり、本人が出演しなくてもリアルな解説動画や多言語翻訳動画が作れる。

// 代表ツール
HeyGen D-ID
// こんな時に使う
  • 研修・説明動画の自動制作
  • 多言語に翻訳した動画を作る
  • YouTubeのトーキングヘッド動画
  • アバターを使ったプレゼン
難易度
無料あり

コーディング

GitHub Copilot

Microsoft/GitHub製のコーディング補助AI。VS Codeなどのエディタに統合され、コードを書きながらリアルタイムで次の行を提案してくれる頼れる相棒。

// 代表ツール
GitHub Copilot VS Code拡張
// こんな時に使う
  • コード補完・自動生成
  • バグの原因を素早く発見
  • コメントから関数を生成
  • 知らない言語のコードを書く
難易度
無料あり

コーディング

Cursor

AIファーストなコードエディタ。プロジェクト全体のコードを理解した上でAIが提案するため、大規模な修正や機能追加が会話だけで完成する次世代ツール。

// 代表ツール
Cursor IDE Claude Code Windsurf
// こんな時に使う
  • 「○○の機能を追加して」と指示するだけ
  • コード全体のリファクタリング
  • エラーメッセージを貼って修正依頼
  • ノーコードに近い感覚で開発
難易度
無料あり

音楽生成

Suno

テキストで曲名とジャンル・歌詞を入力するだけで、ボーカル付きの楽曲を数秒で生成。「爽やかなポップス」「激しいメタル」など自然な言葉で指定できる。

// 代表ツール
Suno Udio
// こんな時に使う
  • 動画のBGM・主題歌を作る
  • オリジナルソングで誕生日を祝う
  • ゲームやアプリの効果音制作
  • プロトタイプ楽曲のアイデア出し
難易度
無料あり
🎙️
音声・読み上げ

ElevenLabs

テキストを驚くほど自然な音声に変換する最高クラスの音声AI。声のクローン機能や感情表現も精度が高く、ポッドキャストや有声本の制作に最適。

// 代表ツール
ElevenLabs Voicevox VOICEPEAK
// こんな時に使う
  • 動画のナレーション自動生成
  • 自分の声のクローンを作る
  • 多言語の音声コンテンツ制作
  • テキストを読み上げて耳で確認
難易度
無料あり

AI検索

Perplexity

「AIを使った検索エンジン」。リアルタイムでウェブを検索し、情報源を明示しながら回答を生成してくれる。Googleの代替として使う人も急増中。

// 代表ツール
Perplexity AI Grok
// こんな時に使う
  • 最新情報を出典付きで調べる
  • 複数のサイトを横断した比較調査
  • 論文や専門記事の要約
  • リアルタイムのニュース確認
難易度
無料あり
